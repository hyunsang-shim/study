========================================================================
    WIN32 응용 프로그램 : DX_3D_STUDY 프로젝트 개요
========================================================================

 상자를 만들고
 
 키보드를 조작하여 이동할 수 있다.
 
 기능:
 - 위치 확인을 위한 기본적인 그리드.
 - 축 확인을 위한 기본적인 GIZMO 표시 (빨간색 = Z축, 파란색 = X 축, 초록색 = Y축)
 - 랜덤 색상의 면을 가지는 육면체
 - 마우스 휠 up = 카메라가 중점을 향해 다가감
 - 마우스 휠 down = 카메라가 중점으로부터 멀어짐
 - 마우스 왼쪽 클릭 + 드래그 = 화면을 마우스 드래그 방향으로 회전시킴
